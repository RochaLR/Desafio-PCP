Desafio para vaga de QA.

1. instalar a dependencias do projeto
```
npm install
```

2. Executar o cypress
 
```
npx cypress open
```

3. Selecionar opção de teste E2E

4. Executar o arquivo de teste 
cypress/e2e/pesquisaProcessos.cy.js


Durante o desafio, o ambiente de homologação (pcphom.com.br) apresentou erro interno ao tentar realizar buscas.
Por isso, os cenários foram validados utilizando o ambiente de produção.
